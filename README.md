A simple habit tracker web application built using Flask and MySQL. 
The application allows users to manage their daily habits and track progress efficiently.

Features:
  User registration and login
  Add new habits
  View existing habits
  Update habit details
  Delete habits
  Track current streak and highest streak
  MySQL database integration
  Simple web interface using Flask templates

Tech Stack:
  Python
  Flask
  MySQL
  HTML
  CSS
  Git & GitHub
